# 📈 Global Stock Tracker (Python)
An interactive financial dashboard built with Python to demonstrate API integration and data visualization.

### 🚀 Key Features
- **Real-Time Data**: Fetches live stock prices using the `yfinance` API.
- **Cross-Exchange Support**: Handles both US (NASDAQ) and Indian (NSE) stocks.
- **Interactive UI**: Built with Streamlit for a seamless user experience.

### 🛠️ Tech Stack
- **Language**: Python
- **Libraries**: Streamlit, Pandas, YFinance
